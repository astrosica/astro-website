Polarimetry and Galactic Magnetism Study Guide

galactic magnetism ; interstellar medium ; polarization ; turbulence

Polarimetry and Galactic magnetism were new and exciting topics for me when I began my thesis work. As I learned about new topics, I added them to a study guide which I could easily refer to later as needed. This document is constantly being updated and covers topics such as phases of the interstellar medium, polarization techniques, probes of Galactic magnetism, magnetohydrodynamics, interstellar turbulence, and more.

Miscellaneous Python Functions

polarimetry ; spectroscopy ; (re-)projections ; FITS files ; Rolling Hough Transform

I have written Python functions for analyzing polarimetric and spectroscopic data, handling and manipulating FITS files, dealing with (re-)projection schemes, and working with the Rolling Hough Transform. While these are mostly written for my own personal use, I am making them publicly available in case anyone finds them helpful. A Github repo for these functions can be found below.
